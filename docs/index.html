SIMPLE_CONSOLE

Console manipulation with colors and cursor control

Overview Quick Start API Reference Colors GitHub

Overview

SIMPLE_CONSOLE provides console manipulation for Eiffel applications on Windows with colored text, cursor control, and screen operations.

Key Features

  • 16 foreground/background colors
  • Cursor positioning
  • Screen/line clearing
  • Console title setting
  • Thread-safe (SCOOP-compatible)

Quick Start

<library name="simple_console" location="$SIMPLE_CONSOLE/simple_console.ecf"/>

Basic Usage

local
    con: SIMPLE_CONSOLE
do
    create con

    -- Print colored text
    con.print_colored ("Success!", con.Green)
    con.print_colored ("Error!", con.Red)

    -- Position and print
    con.set_cursor (10, 5)
    print ("Hello")

    -- Clear screen
    con.clear
end

API Reference

Colors

FeatureDescription
set_foreground (color)Set text color
set_background (color)Set background
set_color (fg, bg)Set both
reset_colorReset to default
print_colored (text, color)Print in color

Cursor

FeatureDescription
set_cursor (x, y)Move cursor
cursor_x, cursor_yGet position
show_cursorMake visible
hide_cursorMake invisible
print_at (text, x, y)Print at position

Screen

FeatureDescription
clearClear screen
clear_lineClear to end of line
